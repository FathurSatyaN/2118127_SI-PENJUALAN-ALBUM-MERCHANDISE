# 2118127_SI-PENJUALAN-ALBUM-MERCHANDISE

File Database Mysql (MK Sistem Informasi)
Tabel Relasi diatas adalah konsep dari SI Penjualan dengan judul yang tertera diatas.
Alur Relasi antar tabel dimulai dari, Tabel Pegawai yang usernamenya akan direlasikan kepada Tabel admin.
Hal tersebut dikarenakan tidak semua pegawai memiliki wewenang sebagai admin (hak akses tertinggi pada database).
Lalu ID_Barang direlasikan pada Tabel Supplier, agar admin dapat mengelola barang apa saja yang perlu di-restock.
